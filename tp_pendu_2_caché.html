Question 1

Créer une fonction qui prend 2 paramètres:

  1. un ensemble (donc de type set) de mots sous forme de str
  2. le nom d'un fichier sous forme de str

La fonction doit charger tous les mots listés dans le fichier dont le nom est spécifié, dans l'ensemble passé en paramètre. Le plus simple comme format du fichier sera d'y mettre un seul mot par ligne.

Vous pouvez utiliser deux méthodes de la classe str:

  • .lower() qui retourne la version d'une chaîne où toutes les lettres sont en minuscule. Rappellez-vous une str est immuable donc la valeur retournée est une NOUVELLE str, l'objet str initial sur lequel on appelle la méthode reste inchangé.
  • .rstrip() qui permet d'enlever un caractère ou une séquence de caractère à droite d'une chaîne.

Elles permettent de créer une copie de l'ensemble des mots, c'est à dire un nouvel objet distinct de l'original, mais qui contient initialement les mêmes éléments. 					import time
					import os

					ENSEMBLE_DE_TOUS_LES_MOTS = set()
					charger_mots_depuis_fichier(ENSEMBLE_DE_TOUS_LES_MOTS, "pendu niv2 liste mots.txt")

						# pause pour permettre au joueur de lire le message avant le prochain nettoyage d'écran
